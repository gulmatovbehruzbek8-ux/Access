To use the bot you need to install Node.js and Antigravity CLI.
First open the terminal on the folder and install libraries by running "npm install".
Before running edit the bot.js file with your notepad and paste your TELEGRAM_BOT_API, TELEGRAM_USER_ID and PROJECT_FOLDER_FULL_DIRECTORY into it.
To run the bot open the terminal on the project folder and  run "node bot.js".
The bot will run as long as that terminal window is not exited.
